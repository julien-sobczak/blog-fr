API : Le nouveau levier de création de valeur

Sans être exceptionnelle, une présentation tout aussi intéressante que son sujet.

Commentaire

Un sujet de grande importance, encore peu répandu mais cela ne serait tarder...

Les intervenants témoignent d'une bonne maîtrise technique (SOAP, REST), et présentent aussi les aspects moins techniques autour du sujet (juridique, etc).

Selon moi, les speakers parviennent à susciter la curiosité sur l'exposition d'API publique mais sans vraiment parvenir à convaincre. Cela reste toutefois un très bon début.

Julien Sobczak
